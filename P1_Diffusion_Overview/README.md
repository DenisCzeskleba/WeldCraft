# P1 Simulate Hydrogen Diffusion

The starting point for most projects. Check out basic 1D, 2D heat transport, hydrogen diffusion thru permeation experiments etc.
I use this a lot to figure out charging times for my samples, estimate saturation,
or, more sinister, when doing reviews, if the the permeation times / Diffusion coefficent even make sense.

Very fast and easy to use VISUAL diffusion (heat transfer too) with a User Interface!
Allows to save animations/videos/Pictures, define Mesh, judge numeric accuracy quantitativly,
explore the mathematical basis for Inflection Point and Time Lag methods for permeation Experiments,
understand the impact of sample thickness, diffusion coefficient, prefill, stepwise measurements, 
temperature, etc etc.
Also allows for different boundary condition (endless plate vs close system for example) so analytical unsolvability of 2D-3D systems can be explored and much more.
